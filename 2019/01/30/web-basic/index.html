      Simple Block 是一个基于线框设计的简单的 Hexo 主题，拆分自 精子的博客。 支持使用 Jade 或 Markdown 来向边栏添加小部件，或向正文前添加横幅。

      技术栈：

      • Jade - 页面模板
      • Less/Sass - 页面样式
      • Bower - 前端包管理器
      • CoffeeScript - Hexo 拓展脚本
      • Gulp - 编译工具
