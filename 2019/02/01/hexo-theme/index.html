After analysing the structure of folders, I realise the theme pack is a npm package.
The main steps for the hexo to change the theme:

  1. read the “_config.yml” file, getting the theme’s name.
  2. go to the theme/theme’s name, and find the npm package.
  3. read the package and generate the contents to the public folder.
  4. Deploy the whole content to the website, coming from the “_config.yml” file.

For the deeper study, it fails to generate when the package has some questions. If so, we need to install this npm first.
